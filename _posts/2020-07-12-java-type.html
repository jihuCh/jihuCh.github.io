--- title: Java Type(타입) author: Jihu Choi layout: post --- 1. 기본 데이터형(Basic data types) 2. 참조 데이터형(Reference data types) __기본 데이터형__ * 숫자형 - 정수형: + Byte, short, int, long(아무것도 쓰지않으면 int로 잡는다.) + 소수점이 없는 정수 + 기본적으로 사용하는 기준: int - 실수형: + float, double + 소수점이 있는 실수, 지수 + 기본적으로 사용하는 기준: double * 논리형 + Boolean + true/false(논리형 literal)(literal = 값) * 문자형 + char + 문자 + '메세지'(단일 따음표 안에 있어야함) * 문자열 + String(문자열 혼자만 대문자를 쓴다) + "string"
